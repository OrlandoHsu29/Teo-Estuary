## 应用介绍

### 1. Teo Recorder (R)
- **路径**: `./teo-recorder/index.html`
- **功能**: 潮汕话录音应用，助力方言文化传承
- **特色**:
  - 拟物风格录音机界面
  - 实时音量监测
  - 录音播放和上传功能
  - API密钥配置

### 2. Teo Translater (T)
- **路径**: `./teo-translater/index.html`
- **功能**: 潮汕话翻译器，支持普通话与潮汕话互译
- **特色**:
  - 扁宽寻呼机拟物风格设计
  - 液晶显示屏效果
  - 双向翻译功能
  - 打字机效果显示
  - LED状态指示灯
  - 字符计数和输入验证

### 3. Teo Phonograph (P)
- **路径**: `./teo-phonograph/index.html`
- **功能**: CD播放器，音频处理与播放工具
- **特色**:
  - 拟物CD播放器设计
  - 动态CD旋转效果
  - 音频文件上传处理
  - 播放控制（播放/暂停/停止/弹出）
  - 音量调节
  - LED状态指示
  - 自动处理和弹出机制

### 4. About (I)
- **功能**: 了解更多关于创意工作室的信息
- **特色**: 项目介绍和技术栈说明

## 技术栈
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **字体**: Google Fonts + 本地字体
- **样式**: CSS Grid, Flexbox, CSS动画
- **交互**: 原生JavaScript，无框架依赖
- **音频**: Web Audio API (用于音效反馈)
- **振动**: Vibration API (设备震动反馈)
- **文件处理**: FileReader API, URL.createObjectURL

## 键盘快捷键

### 主选择页面 (index.html)
- `Enter`: 快速开始（进入录音应用）
- `1-4`: 数字键选择对应应用

### 翻译应用 (Teo Translater)
- `Enter`: 翻译文本
- `Escape`: 清除文本
- `↑/↓`: 滚动文本区域

### 录音应用 (Teo Recorder)
- `空格键`: 快速开始/停止录音

### CD播放器 (Teo Phonograph)
- 主要通过界面按钮操作

## 功能说明

### Teo Translater 翻译功能
- **翻译方向**: 支持普通话↔潮汕话双向翻译
- **输入限制**: 最多100字符，实时计数显示
- **翻译效果**: 打字机效果逐字显示结果
- **状态指示**: LED灯显示系统状态
- **音频反馈**: 翻译完成时的音效提示
- **设备震动**: 支持震动设备的触觉反馈

### Teo Phonograph CD播放功能
- **音频上传**: 支持所有音频格式，最大50MB
- **自动处理**: 模拟音频文件处理过程
- **CD动画**: 载入、旋转、弹出动画效果
- **播放控制**: 完整的播放/暂停/停止/弹出功能
- **音量控制**: 实时音量调节
- **自动弹出**: 播放完成后自动弹出CD
- **状态显示**: 实时显示播放状态和进度

## 开发说明

### 添加新应用
1. 在对应目录创建应用文件夹
2. 更新 `index.html` 中的选择卡片
3. 遵循拟物设计风格

### 样式定制
- 每个应用使用独立的CSS文件
- 拟物风格的变量定义在各自CSS文件顶部
- 动画效果使用CSS @keyframes

### 交互功能
- 每个应用使用独立的JavaScript文件
- 全局功能通过window对象暴露
- 事件委托优化性能
